# Mobile Phone Repair Portfolio

## Overview

This repository contains multiple hands-on smartphone repair and teardown projects.

Each project demonstrates a structured troubleshooting approach, safe hardware handling, and clear technical documentation.

The goal is to simulate real-world workflows used in IT support and hardware roles.

---

## Projects

### Samsung Galaxy S10 5G
- Full teardown and internal component removal  
- Battery isolation and removal  
- Motherboard extraction  
- Reassembly validation  

(more devices will be added)

## Skills demonstrated

- Hardware troubleshooting and diagnostics  
- Mobile device disassembly and reassembly  
- Risk management (battery and fragile components)  
- Structured step-by-step workflows  
- Technical documentation
